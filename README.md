# MERN Stack Application

## Project Overview
This is a MERN stack application that combines MongoDB, Express.js, React.js, and Node.js to build a full-stack web application. The MERN stack is a popular technology stack for building dynamic websites and applications, providing a solid foundation for both frontend and backend development.

## Features
- **Full-Stack Development**: Both client-side and server-side code are included in this project, demonstrating how to interact with APIs and manage state effectively.
- **Responsive Design**: Utilizes React components that adapt to different screen sizes, providing a seamless user experience across devices.
- **CRUD Operations**: Implemented Create, Read, Update, and Delete functionalities for managing data.
- **User Authentication**: Secure user authentication and authorization for controlled access to various parts of the application.
- **RESTful APIs**: Well-structured APIs build interaction between frontend and backend.

## Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/Tribikram11/mern-app.git
   cd mern-app
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Set up the environment variables**
   Create a `.env` file and configure your MongoDB URI, JWT secret, and any other required variables.
4. **Run the Application**
   ```bash
   npm run dev
   ```
   The application will start running on `http://localhost:3000` for the client and `http://localhost:5000` for the server.

## Technologies Used
- **MongoDB**: NoSQL database for storing application data.
- **Express.js**: Backend web framework for Node.js, simplifying the server setup and routing.
- **React.js**: Frontend library for building user interfaces with reusable components.
- **Node.js**: JavaScript runtime for server-side development.
